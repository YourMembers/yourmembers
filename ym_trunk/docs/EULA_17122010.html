This is a pretty standard thing for software licenses and covers the Coding Futures EULA agreement and whilst in theory we can revoke your license at any time for any reason, it’s nothing something we’re likely to ever do. The website may have multiple domain names or be served from multiple servers (real or virtualized), as long as it is serving up the same content on them. If the content differs by domain or server, it is classed as a different site.

 

For single licenses, we do make an exception where if the software is first installed on a test site during development, but then used on a different site when the product is put live, on the understanding that the software is used in the test environment ONLY before it is used in the live environment and no transactions (other than obvious test transactions) are made on the test site.
