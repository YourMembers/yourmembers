Your Members

This project contains the latest trunk files of the Your Members WordPress Membership plugin, it is primarily designed as an archive for existing customers. We do not recommend people use this plugin and it is for archive purposes only.

For more information see http://blog.yourmembers.co.uk/2013/end-of-your-members/

WARNING

This software is supplied as is, with no warranty please read the readme before installing. We cannot offer support on this software and will not be making further additions or bug fixes currently.

FAQ

Why did you stop developing? We could no longer afford to actively develop a new version and provide support for existing version to a level we felt comfortable with. In the end we were failing to do both to a standard that our customers expected.

Is it usable? Yes. At the time of writing it was stable, we would however recommend looking at alternatives

Can I develop it? Sure but we wouldn't recommend it, the plugin has grown "organically" and is pretty much a guide to how not to design a modern WordPress plugin.

Will you develop feature x for me? Sorry :( but we have to draw a line and it was an all or nothing approach so I'm afraid we are unable to offer support and customisation.

Can you recommend a WordPress Membership Management Plugin? Not really, but the closest in terms of features is S2Pro
